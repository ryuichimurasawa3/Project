## S1｜エグゼクティブサマリー

**取締役会を本当に変えるのは、実効性評価ではなく、指名と報酬の再設計である。**

日本企業の大多数が取締役会実効性評価を実施するようになった。しかし、改革が定着している企業は少ない。

その根本原因は明確だ。

> **「誰をボードに置くか（指名）」と「何に対して報いるか（報酬）」が変わっていない。**

この2つが変わらない限り、実効性評価の提言は執行されず、ボードは形を変えながら実態を保ち続ける。

WTWは、経営者報酬と人材のプロフェッショナルとして、**指名・報酬・ボード構成の3領域を統合設計できる唯一のファーム**として、貴社のBoard Transformationを支援する。

## S2｜なぜ取締役会は変わらないのか

**評価が変えられるのは「議論の質」だけだ。構造は変えられない。**

実効性評価が普及して10年が経つ。それでも多くの企業のボードが変わらない理由は、評価のターゲットが間違っているからではなく、**評価だけでは動かせない「構造」がある**からだ。

### 取締役会の構造を規定する2つの要素

**① 指名：誰がボードにいるか**

- スキルマトリクスが形式的で、実際の候補者選定は属人・非公式
- 後継者計画が整備されておらず、スキルの断絶が繰り返される
- 結果、ボード構成は「変えたくても変えられない」状態が続く

**② 報酬：何のために動くか**

- CEOの評価指標が短期業績に集中している限り、長期・ESG視点は軽視される
- 報酬委員会の独立性が形式的なままでは、経営者報酬の設計が機能しない
- 結果、取締役会が「監督機能」を果たすための動機づけが壊れる

### 実効性評価との関係

| 役割 | 実効性評価 | 指名・報酬の再設計 |
|------|-----------|-----------------|
| できること | 議論の質・プロセスを診断する | ボードの構造を変える |
| できないこと | 構成・報酬体系を変える | ─ |
| 位置づけ | 達成度の確認手段 | 変革の本体 |

**実効性評価は、改革の「スタート」でも「ゴール」でもない。指名と報酬の再設計が実装された後に、達成度を測る手段として機能する。**

## S3｜Board Transformationの3レバー

**「何を変えるか」ではなく「構造のどこに手を入れるか」が問題だ。**

取締役会の変革は、3つのレバーを統合的に動かすことで実現する。

```
        ┌─────────────────────────────┐
        │     あるべきボード像の定義    │
        └──────────────┬──────────────┘
                       │
        ┌──────────────┼──────────────┐
        ↓              ↓              ↓
  ┌──────────┐  ┌──────────┐  ┌──────────┐
  │  指名・  │  │  報酬体系 │  │  構成・  │
  │サクセッシ│  │  の戦略  │  │  運営    │
  │  ョン   │  │  連動化  │  │  設計    │
  │        │  │         │  │         │
  │誰がボード│  │何のために│  │どう議論し│
  │を担うか  │  │報いるか  │  │決めるか  │
  └──────────┘  └──────────┘  └──────────┘
        │              │              │
        └──────────────┼──────────────┘
                       ↓
              ┌─────────────┐
              │  達成度評価  │
              └──────┬──────┘
                     ↓
              次のボード像へ（サイクル）
```

### 3レバーの相互依存性

3つのレバーは独立していない。

- 指名が変われば、ボードに入るスキル・視点が変わる
- 報酬が変われば、経営の優先順位と監督の焦点が変わる
- 構成・運営設計が変われば、議論の深度と意思決定の質が変わる

**どれか1つだけを変えても、変革は定着しない。** WTWは3つのレバーを同じチームで統合設計できる。

## S4｜WTW Board Solution のフルサイクル

**「改革設計」から「達成度評価」まで、一気通貫で支援する。**

### フェーズ構成

**Phase 1｜改革フェーズ**（ボード構造を変える）

対象：先進的ボード体制を目指すが、どこから始めるかが不明確な企業

| 支援内容 | 具体的なアウトプット |
|---------|------------------|
| あるべきボード像の定義 | ボードビジョン・設計原則 |
| ボードスキルマトリクスの設計 | スキル要件の可視化・ギャップ分析 |
| 社外取締役の選定基準・プロセス構築 | 候補者評価フレームワーク |
| 取締役会規程・運営ルールの見直し | 改訂規程・運営ガイドライン |
| 指名委員会の機能強化 | プロセス設計・委員会運営支援 |
| 経営者報酬体系の戦略連動化 | 報酬ポリシー・KPI連動モデル |

**Phase 2｜達成度評価フェーズ**（改革の深度を測り、次に進む）

対象：社外取締役過半数・独立委員会設置を実現し、次の変革を目指す企業

| 支援内容 | 具体的なアウトプット |
|---------|------------------|
| ゴール対比の達成度評価設計 | 評価指標・測定フレームワーク |
| 報酬体系の高度化 | 長期インセンティブ・ESG連動設計 |
| サクセッションプランの高度化 | 後継者育成ロードマップ |
| 株主・ステークホルダー対話支援 | エンゲージメント戦略 |

**Phase 3｜次のゴール設定**（サイクルの継続）

達成度評価の結果を踏まえ、次のボード像を再定義し、改革サイクルを継続する。

## S5｜診断：貴社のボードはどこから始めるか

**変革の入り口は企業ごとに異なる。まず現在地を特定する。**

以下の問いに答えることで、WTWが最初に着手すべき領域が明確になる。

### ボード診断チェックリスト

| # | 問い | Yes / No |
|---|-----|---------|
| 1 | 社外取締役が取締役会の過半数を占めているか？ | |
| 2 | 指名・報酬委員会が独立委員主導で設置されているか？ | |
| 3 | 「あるべきボード像」を文書として定義しているか？ | |
| 4 | 経営者報酬が中長期戦略・ESG目標と連動して設計されているか？ | |
| 5 | CEO・役員のサクセッションプランが整備されているか？ | |

### 診断結果と推奨される入り口

**0〜2個：改革設計から着手**
ボード像の定義と構成・委員会の抜本的見直しが最優先。まず「何を目指すか」を定義することから始める。

**3〜4個：改革の実装・高度化フェーズ**
形式的な整備は進んでいるが、実質が伴っていない領域がある。指名・報酬の質的高度化が次のステップ。

**5個すべて：達成度評価とサイクルの高度化**
先進的ボード体制を実現済み。「どこまでできているか」を測定し、次のボード像を設定する段階にある。

> **どのフェーズにいても、WTWは最適な入り口から支援を開始できる。**
> 診断セッション（1〜2時間）で現状と優先課題を特定し、スコープをご提案する。

## S6｜なぜWTWか

**報酬と人材のプロフェッショナルだからこそ、ボードを変えられる。**

ガバナンスアドバイザリーを提供するファームは多い。しかし、取締役会を「構造から変える」ために必要なケイパビリティを一つのチームで持つファームは、ほとんど存在しない。

### WTWの差別化ポイント（他社との比較）

**① 経営者報酬の設計力と市場データ**

WTWは世界最大級の経営者報酬ベンチマークデータベースを保有し、国内外の報酬水準を実データで比較できる唯一のファームだ。「市場水準に準拠した」という抽象的な説明ではなく、具体的な数値根拠に基づいて報酬体系を設計する。

純粋なガバナンスコンサルや法律事務所には、この報酬設計力はない。

**② 人材戦略のトータルカバー**

WTWは経営層だけでなく、組織全体の人材・報酬戦略を手がける。これにより、ボードレベルで決定した報酬戦略が現場の人材マネジメントと整合しているかを一気通貫で見ることができる。

「ボードが決めたことが組織に浸透しない」という問題の根本を、組織全体の人材戦略として解決できる。

**③ 利益相反のない独立性**

サーチファーム（Korn Ferry、Spencer Stuartなど）は、候補者の斡旋から収益を得る構造上、指名プロセスの独立した設計には利益相反が生じる。WTWは候補者を紹介することから収益を得ないため、**純粋に企業側の利益を優先した指名プロセスの設計・評価**ができる。

**④ 指名×報酬×評価の統合設計**

通常、指名支援・報酬設計・実効性評価はそれぞれ別のファームが担う。WTWはこの3領域を同じチームで統合設計できるため、「各領域の提言が矛盾する」「フェーズをまたぐたびに情報が失われる」という問題が起きない。

**⑤ グローバル知見×日本の実務経験**

日本のコーポレートガバナンス改革の文脈を理解しながら、グローバル企業のボード設計事例を直接参照できる。海外機関投資家の期待水準と国内の実務慣行を両立した設計が可能だ。

### 一言でいえば

> **WTWは「ガバナンスを知っている」だけでなく、「人と報酬で組織を変えてきた」ファームだ。**
> だからこそ、取締役会という組織の頂点を、構造から変えることができる。

```
│ ・課題の特定と優先論点の整理

│ ・指名プロセス・評価基準の構築

Phase 3（年次）: 達成度評価
  ・ゴール対比の評価実施
  ・提言と次のボード像の再設定
  ・改革サイクルの継続
```

### WTW支援チームの構成

**WTWが提供する体制は、他のファームでは組成できない。**

| 役割 | WTWのケイパビリティ |
|-----|------------------|
| Board Advisory リード | ガバナンス設計・取締役会改革の統括 |
| 経営者報酬専門家 | グローバル報酬データ×設計の実務 |
| 指名・サクセッション専門家 | 指名プロセス・後継者計画の実務 |
| 人材戦略アドバイザー | 組織全体の人材マネジメントとの接続 |

このチーム構成が意味すること：

**報酬の専門家がボードを設計する。** 「報酬はどう設計するか」を知っているプロが指名・構成の設計を担うため、「形式的なガバナンス整備」ではなく「機能する構造の変革」ができる。

**人材のプロが、ボードと組織をつなぐ。** WTWは取締役会と現場組織の両方を支援しているファームだ。ボードで決定したことが、経営戦略を通じて組織全体に実装されるところまで一気通貫で見ることができる。

## S8｜WTW Board Solution サービスメニュー

**指名と報酬を核に、ボード改革に必要なすべてをカバーする。**

### カテゴリーA｜指名・サクセッション

取締役会の「構成」を変え、ボードに必要なスキルと多様性を実現する。

- ボードスキルマトリクスの設計・ギャップ分析
- 社外取締役の選定基準・評価プロセスの構築
- 指名委員会の機能強化・プロセス設計
- CEOサクセッションプランの策定
- 後継者候補の育成ロードマップ設計
- 取締役個人評価の設計・実施支援

### カテゴリーB｜経営者報酬

経営の優先順位を変え、ボードによる監督機能を実質化する。

- 報酬ポリシーの策定・見直し
- 報酬体系の中長期戦略・ESG連動設計
- 報酬水準のグローバルベンチマーキング
- 長期インセンティブ（株式報酬）の設計
- 報酬委員会の継続的アドバイザリー支援
- 有価証券報告書・統合報告書の報酬開示支援

### カテゴリーC｜取締役会の実効性・構成・運営

ボードの「議論の質」と「意思決定の構造」を変える。

- あるべきボード像の定義（ボードビジョン策定）
- 取締役会実効性評価（第三者評価）
- 達成度評価（ゴール対比型）
- 取締役会規程・運営ガイドラインの見直し
- Non-executive Chairman・筆頭社外取締役の役割設計
- 株主エンゲージメント戦略の策定・対話支援

## S9｜次のステップ

**まず「診断セッション」から始める。スコープと費用は、その後に決める。**

貴社の現状と課題を把握しなければ、最適な支援の入り口を提案できない。WTWは必ず現状診断から始める。

### ステップ1｜現状診断セッション（1〜2時間）

以下を確認し、優先課題と着手領域を特定する。

**確認事項（例）：**

- 現在の取締役会構成・委員会設計の概要
- 実効性評価の実施状況と主要な課題認識
- 指名・報酬の現行プロセスと改善余地
- 機関投資家・株主からのフィードバック
- 経営戦略との接続における論点

### ステップ2｜課題マップと支援スコープのご提案（1〜2週間以内）

診断セッションを踏まえ、WTWから以下を提示する。

- 優先して取り組むべき領域と理由
- 推奨する支援スコープ（指名・報酬・評価のどれから、どの順で）
- 進め方・体制・期間の大枠

### ステップ3｜着手領域と進め方の合意

どの領域から始めるか、どの深度で取り組むかを合意し、Phase 0（現状診断・ボード像の仮説設定）へ。
