## S1｜エグゼクティブサマリー

**KEY MESSAGE：**
> 実効性評価を「達成度の物差し」として機能させる前提は、ボードの構造変革だ。
> WTWは指名・報酬・取締役会の実効性、3つのレバーから、その変革をトータルに設計する。

多くの企業が取締役会実効性評価を継続してきた。しかし評価が改革につながらない場合、問題は評価の手法にあるのではない。評価が計測すべき「構造的変化」がまだ起きていないことにある。

実効性評価が本来の力を発揮するのは、目指すボード像が定義され、その実現に向けた変革が動き出した後だ。「今どこにいるか」ではなく「目標にどれだけ近づいたか」——この問いに答えて初めて、評価は改革の推進力になる。

取締役会の構造を変えるレバーは3つある。**誰をボードに置くか（指名・サクセッション）、何に対して報いるか（経営者報酬）、ボードをいかに実効的に機能させるか（取締役会の実効性）**。どこから着手するかは企業の現状と課題によって異なる。

WTWは経営者報酬と人材のプロフェッショナルとして、この3つのレバーすべてに専門性を持つ。まず貴社のボードの現在地を診断し、最適な変革の入り口と進め方をともに設計する。

## S2｜なぜ実効性評価は機能しなくなるのか

**評価の手法が悪いのではない。会社のガバナンスのステージに合わせて、評価のあり方を進化させていないことが問題だ。**

取締役会実効性評価の導入初期は、定型的なフレームワークで十分に機能する。しかし多くの企業で、一定の年数を経た後にこんな声が上がる——**「毎年同じ課題が繰り返される」「評価しているのに実効性が向上している実感がない」「プロセスが形式化している」**。

これは評価の失敗ではなく、**進化のサインだ。** ボードが成熟するにつれ、「今どこにいるか」を確認する評価から、「目標にどれだけ近づいたか」を測る評価へとアップデートが求められる。その移行が起きていないとき、評価はマンネリ化する。

### 評価が機能しなくなる3つの根本原因

**① 「目指すボード像」が定義されていない**
ゴールが明確でなければ、評価は現状確認の儀式になる。「どこを目指しているか」が定義されて初めて、評価は達成度の物差しとして機能する。

**② 指名・サクセッションの構造が変わっていない**
評価で「スキルの多様性が不足している」と指摘されても、候補者選定のプロセスが属人的・非公式なままでは翌年の構成は変わらない。

**③ 報酬体系が戦略と連動していない**
評価で「長期・ESG視点の強化が必要」と提言されても、経営者の評価指標が短期業績に集中している限り、経営の優先順位は変わらない。

> 評価が達成度の物差しとして機能するためには、「何を達成するか」を定義し、それを実現する構造に先に手を入れる必要がある。

## S3｜Board Transformationの3レバー

**構造を変えなければ、議論を変えても取締役会は変わらない。**

あるべきボード像を定義した上で、その実現に向けて3つのレバーを統合的に動かす。

```
┌─────────────────────────────────┐
│     あるべきボード像の定義        │
└────────────────┬────────────────┘
                 │
     ┌───────────┼───────────┐
     ↓           ↓           ↓
 ┌────────┐  ┌────────┐  ┌────────┐
 │  指名・  │  │ 経営者  │  │取締役会 │
 │サクセッ  │  │  報酬   │  │ の実効性 │
 │ション   │  │        │  │        │
 │        │  │        │  │        │
 │誰がボード│  │何に対して│  │いかに実 │
 │を担うか  │  │報いるか  │  │効的に   │
 │        │  │        │  │機能するか│
 └────────┘  └────────┘  └────────┘
     └───────────┼───────────┘
                 ↓
        ┌────────────────┐
        │   達成度評価    │
        └───────┬────────┘
                 ↓
        次のボード像へ（サイクル）
```

**① 指名・サクセッション**
ボード像に基づいて必要なスキル・多様性を定義し、客観的・公式な選定プロセスと後継者計画を整備する。属人的な選定から、構造的な仕組みへ。

**② 経営者報酬**
中長期戦略・ESG目標と連動した報酬体系を設計する。経営者が何で評価されるかが、取締役会による監督の実質を決める。

**③ 取締役会の実効性**
アジェンダ設計・情報フロー・委員会運営・規程管理など、ボードが実効的に機能するための基盤を整備する。海外ではコーポレートセクレタリーが担うこの機能が、日本では取締役会事務局の役割として改めて問われている。

### 3レバーは独立していない

ボード像が定義されて初めて、指名の基準が生まれる。指名が変わって初めて、ボードの監督機能が変わる。報酬が変わって初めて、経営の優先順位が変わる。**1つだけ動かしても、変革は定着しない。**

3つのレバーが動き出した後、実効性評価は「達成度の物差し」として機能する。その結果が次のボード像の設定に活かされ、改革サイクルが継続する。

## S4｜実効性評価を「達成度評価」として再定義する

**評価は、改革サイクルの「終点」ではなく「測定点」だ。**

### 現状：評価がサイクルから切り離されている

```
実効性評価 → 提言書の作成 → 翌年また同じ課題 → 実効性評価…
```

評価がボード像の定義とも、3つのレバーへの介入とも切り離されている状態では、評価は改革の力にならない。

### WTWのアプローチ：評価をサイクルに組み込む

```
ボード像の定義 → 3レバーの変革 → 【達成度評価】→ 次のボード像の設定 → …
```

実効性評価が「達成度評価」として機能するための3つの条件：

**① ゴールが定義されている**
自社が目指すボード像が明確でなければ、達成度は測れない。一般的なベストプラクティスとの比較では、自社の改革の進捗は見えない。

**② 評価の前に変革が動いている**
3つのレバーが動いていなければ、評価は前年と同じ現状を確認するだけになる。

**③ 評価結果が次のゴールに接続される**
評価は終点ではなく、次のボード像を設定するための起点だ。提言が翌期の改革アクションに連動して初めて、サイクルが回る。

| | 従来型の実効性評価 | 達成度評価（WTWのアプローチ） |
|--|---|---|
| 参照軸 | 一般的なベストプラクティス | 自社が定義したボード像 |
| 評価の位置づけ | 現状スナップショット | 改革の進捗測定 |
| 評価後のアクション | 提言書 → 次期評価 | 提言 → 次のボード像の再設定 |

> 評価を「達成度の物差し」として機能させることが、Board Transformationを一時的な改革で終わらせない仕組みをつくる。

## S5｜診断：どのレバーから変革を始めるか

**どこから着手するかは、3つのレバーごとに現在地を確認することで見えてくる。**

### ステップ1：レバーごとに現在地をチェックする

**① 指名・サクセッション**

| | チェック項目 |
|--|--|
| 形式 | 指名委員会が設置され、独立社外取締役が過半数を占めているか |
| 実質 | スキルマトリクスが候補者の実際の選定に活用されているか |
| 実質 | 社外取締役の選定プロセスが明文化・公式化されているか |
| 実質 | CEOおよび主要役員のサクセッションプランが整備されているか |

**② 経営者報酬**

| | チェック項目 |
|--|--|
| 形式 | 報酬委員会が設置され、独立社外取締役が過半数を占めているか |
| 実質 | 経営者報酬が中長期の戦略目標・ESG指標と連動して設計されているか |
| 実質 | 報酬水準が市場データをもとに定期的に検証されているか |

**③ 取締役会の実効性**

| | チェック項目 |
|--|--|
| 形式 | 社外取締役が取締役会の過半数を占めているか |
| 実質 | あるべきボード像が定義・文書化されているか |
| 実質 | アジェンダ設計と情報提供が実効的な議論を支えているか |

### ステップ2：レバーごとの現在地と推奨される入り口

| レバー | 未整備（形式なし） | 形式のみ整備 | 実質も機能している |
|---|---|---|---|
| **指名・サクセッション** | 委員会設計・選定プロセスの構築から | スキルマトリクス・サクセッションの実質化 | 高度化・達成度評価へ |
| **経営者報酬** | 報酬体系・委員会の設計から | 戦略連動・市場水準との整合化 | 高度化・達成度評価へ |
| **取締役会の実効性** | ボード像の定義・基盤整備から | アジェンダ・情報フローの実質化 | 高度化・達成度評価へ |

> 最も「未整備」または「形式のみ」に近いレバーが、変革の最優先着手点だ。
> 複数のレバーに課題がある場合は、診断セッションで優先順位を一緒に見極める。

## S6｜なぜWTWか

**経営者報酬と人材のプロフェッショナルファームだからこそ、ボードを構造から変えられる。**

### ① 指名・サクセッション：ガバナンス特化の専門性 × 組織人事の融合

総合コンサルや人材サーチファームとは異なり、WTWはガバナンスの観点と組織人事の専門性を融合させた独自のアプローチで指名・サクセッションを支援する。

組織人事のプロフェッショナルファームとして、**従業員エンゲージメントや投資家視点など「客観的な声」を実効性評価に組み込むノウハウ**を保有。多面的な実効性検証を通じて、**ガバナンス（指名・報酬）と組織人事（エンゲージメント・文化）の両面から、実践へと結び付ける**。

また、候補者の推薦・斡旋から収益を得るサーチファームと異なり、WTWは指名プロセスの設計・評価において完全に独立した立場を保つ。

### ② 経営者報酬：国内トップクラスのデータ基盤 × 圧倒的な実装力

WTWは大手上場企業を中心とした国内有数の「経営者報酬データベース」を保有し、国内外の最新インサイトを踏まえた信頼性の高い情報を分析・提供する。

約25年間で構築されたこのコミュニティは、**日本のトップ300社の70%以上をカバー（国内トップクラス）**し、90%以上の高い継続参加率により経年分析の安定性を保持している。単なる情報データベースではなく、ノウハウと価値を共有する基盤だ。

さらに多くの指名・報酬委員会の支援実績から、**取締役会の提言を役員報酬制度や後継者計画という経営基盤へと具体的に実装させる知見**を有する。取締役会が高い視座で議論しても、それを執行に落とし込むことは容易ではない。その実装力こそが、WTWの本質的な価値だ。

### ③ 一体組織としての統合設計力：変革の全体をマネジメントする

WTWが真に差別化できる点は、「取締役会（ガバナンス）→ 指名・報酬 → 人的資本経営 → 組織文化・エンゲージメント」という**変革の連鎖全体を、一つのチームで統合的にマネジメントできること**だ。

他のファームは「指名専門」「報酬専門」「評価専門」に分断されており、各論をつなぐ視点を持たない。WTWは経営者報酬とボードアドバイザリーが一体となった組織として、取締役会の意思決定が組織の末端まで実装される変革を、一つのチームで統合的に支援できる。

## S7｜進め方・体制

**3レバーそれぞれに専門家を置き、一体チームとして統合設計する。**

### 支援体制：3レバー × 専門家の対応

| 役割 | 担当レバー | 主な支援内容 |
|---|---|---|
| **プロジェクトリード**（統括） | 全体統合・変革の統括 | ボード像の定義、3レバーの統合設計、クライアント窓口 |
| **指名・サクセッション専門家** | レバー① 指名 | 指名プロセス設計、スキルマトリクス、後継者計画 |
| **経営者報酬専門家** | レバー② 経営者報酬 | 報酬体系設計、ベンチマーク分析、報酬委員会支援 |
| **Board Advisory** | レバー③ 取締役会の実効性 | アジェンダ設計、達成度評価、規程整備 |

必要に応じ、人的資本・エンゲージメント専門家と連携（組織文化・従業員エンゲージメント測定）

**これが唯一無二である理由：** 報酬・指名・評価の専門家が同一チームで動くため、3レバーの設計が整合する。各領域を別ファームに発注した際に生じる「提言の矛盾」「フェーズをまたぐ情報の分断」が起きない。

### 進め方のロードマップ

**Phase 0｜現状診断・ボード像の仮説設定**（1〜2ヶ月）

- 主要役員インタビュー（取締役会議長・指名委員長・報酬委員長・CEOなど）
- 現行規程・委員会資料・報酬設計のレビュー
- 課題の特定と優先論点の整理
- アウトプット：診断レポート・「あるべきボード像」の仮説

**Phase 1｜改革設計**（3〜6ヶ月）

- ボードビジョンの策定と合意形成（経営陣・取締役会）
- 3レバーのうち優先領域から設計に着手
- アウトプット：改革設計書・スキルマトリクス・報酬体系設計・規程改訂案

**Phase 2｜実装・巡航運用**（継続）

- 指名委員会・報酬委員会のファシリテーション・アドバイザリー
- 規程・ガイドラインの改訂支援
- 株主エンゲージメント対応支援

**Phase 3｜達成度評価**（年次）

- ボード像に対する達成度の評価
- 提言と次のボード像の再設定 → サイクルを継続

## S8｜WTW Board Solution サービスメニュー

**指名と報酬を核に、ボード改革に必要なすべてをカバーする。**

### カテゴリーA｜指名・サクセッション

取締役会の「構成」を変え、ボードに必要なスキルと多様性を実現する。

- ボードスキルマトリクスの設計・ギャップ分析
- 社外取締役の選定基準・評価プロセスの構築
- 指名委員会の機能強化・プロセス設計
- CEOサクセッションプランの策定
- 後継者候補の育成ロードマップ設計
- 取締役個人評価の設計・実施支援

### カテゴリーB｜経営者報酬

経営の優先順位を変え、ボードによる監督機能を実質化する。

- 報酬ポリシーの策定・見直し
- 報酬体系の中長期戦略・ESG連動設計
- 報酬水準のグローバルベンチマーキング
- 長期インセンティブ（株式報酬）の設計
- 報酬委員会の継続的アドバイザリー支援
- 有価証券報告書・統合報告書の報酬開示支援

### カテゴリーC｜取締役会の実効性

ボードの「議論の質」と「意思決定の構造」を変える。

- あるべきボード像の定義（ボードビジョン策定）
- 取締役会実効性評価（第三者評価）
- 達成度評価（ゴール対比型）
- 取締役会規程・運営ガイドラインの見直し
- Non-executive Chairman・筆頭社外取締役の役割設計
- 株主エンゲージメント戦略の策定・対話支援

## S9｜次のステップ

**まず「診断セッション」から始める。支援スコープと進め方は、診断結果をもとに設計する。**

貴社の現状と課題を把握しなければ、最適な支援の入り口を提案できない。WTWは必ず現状診断から始める。

### ステップ1｜現状診断セッション（1〜2時間）

以下を確認し、優先課題と着手領域を特定する。

**確認事項（例）：**

- 現在の取締役会構成・委員会設計の概要
- 実効性評価の実施状況と主要な課題認識
- 指名・報酬の現行プロセスと改善余地
- 機関投資家・株主からのフィードバック
- 経営戦略との接続における論点

### ステップ2｜課題マップと支援スコープのご提案（1〜2週間以内）

診断セッションを踏まえ、WTWから以下を提示する。

- 優先して取り組むべき領域と理由
- 推奨する支援スコープ（指名・報酬・評価のどれから、どの順で）
- 進め方・体制・期間の大枠

### ステップ3｜着手領域と進め方の合意

どの領域から始めるか、どの深度で取り組むかを合意し、Phase 0（現状診断・ボード像の仮説設定）へ。
