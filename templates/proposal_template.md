### 2. エグゼクティブサマリー（1枚）

**KEYメッセージ**（1〜2文で完結）：
> [例：現行の取締役会構成では、独立性と多様性の両面で機関投資家の期待水準を下回っており、早期の構造的対応が必要です。]

1. 現状：[課題の本質]

### 3. 現状認識・課題設定（2〜3枚）

#### 3-1. 外部環境・規制動向
- [関連するコーポレートガバナンス・コードの改訂動向]
- [機関投資家・議決権行使方針の変化]
- [同業他社・市場の動き]

#### 3-2. クライアントの現状
- [現行体制の概要]

> **ポイント**：課題は「〜が不足している」ではなく「〜が[具体的影響]を引き起こしている」と因果で表現する

### 4. 論点・分析（2〜4枚）

#### 論点の構造（MECEで整理）

```
論点A: [例：取締役会の構成]
  └ A-1: 独立社外取締役の比率
  └ A-2: スキル・マトリクスのギャップ
  └ A-3: 多様性（ジェンダー・国際性）

論点B: [例：委員会の実効性]
  └ B-1: 指名委員会の独立性
  └ B-2: 報酬委員会の設計
```

#### 各論点の記載フォーマット

## ファイル命名規則

```
YYYYMMDD_[クライアント略称]_[テーマ略称]_[ステータス].pptx

例：
20260115_FujiMediaHD_取締役会実効性評価_Draft.pptx
20260201_FujiMediaHD_取締役会実効性評価_Final.pptx
```

## ステータス定義

| ステータス | 意味 |
|---|---|
| Draft | 初稿（内部レビュー前） |
| Review | レビュー中 |
| Final | クライアント提出版 |
